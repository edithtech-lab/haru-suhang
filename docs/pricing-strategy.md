# 하루수행 — 수익화 전략

작성일: 2026-04-30
업데이트: TBD (사용자 피드백 후)

## 핵심 원칙

1. **명상 앱 = 영적 도구.** 강한 수익화는 거부감을 부른다.
2. **출시 후 최소 3개월은 무료.** 신뢰 형성 전에 결제 깔면 이탈↑.
3. **핵심 수행 (108배·명상·염불)은 영원히 무료.** 잠그면 앱 정체성이 무너짐.
4. **AI·콘텐츠·클라우드** 같은 "더 깊이 가고 싶은 사람" 영역만 유료화.
5. **B2B 사찰 라이센스가 진짜 수익 엔진** — Phase 4의 핵심.

## 시장 비교 — Plum Village는 왜 무료인가

| 앱 | 모델 | 비고 |
|---|---|---|
| **Plum Village** | 100% 무료 + 자원봉사 + 기부 | 사찰 본부 운영 자금 (수도원·리트릿) |
| **코끼리** | 9,900원/월 | 한국 시장 표준 가격, conversion 1~3% |
| **마보** | 9,900원/월 | 코끼리와 유사 |
| **Calm/Headspace** | 월 $14.99 + 콘텐츠 분리 | 글로벌 |

→ haru-suhang은 **자체 기관(사찰)이 없으니** 100% 무료는 어려움. 그러나 9,900원 한국 시장 표준은 너무 비쌈 (4,900원이 적정).

## Phase 별 로드맵

### Phase 1 (0~3개월): 무료 + 보시
- **모든 기능 무료**, 결제 X
- 헤더 또는 설정에 "🙏 보시하기" 버튼 (1,000 / 3,000 / 5,000원 / 자유금액)
- 토스 일반결제 위젯
- 목표: 신뢰 형성, 사용자 확보

### Phase 2 (3~9개월): 마이크로 결제
| 상품 | 가격 | 비고 |
|---|---|---|
| 사찰 사운드 팩 | 1,900원 | 단품 (목탁·종 고음질·자연 소리) |
| 경전 AI 낭독 (1편) | 2,900원 | 반야심경·천수경 단품 |
| AI 법현 스님 1주일권 | 1,900원 | 한정 무제한 |
| AI 법현 스님 1개월권 | 4,900원 | 1일 3회 → 무제한 |
| 보시 (월정기) | 자유금액 | 매월 자동 |

### Phase 3 (9~18개월): 멤버십 정식 도입
- **수행자 멤버십**: 월 4,900원 / 연 39,000원 (33% 할인)
- 포함:
  - AI 법현 스님 무제한
  - 모든 경전 낭독 (천수경·금강경·신심명·금강경 등)
  - 명상 가이드 라이브러리 (호흡·자비·보디스캔 5/10/20분)
  - 마음기록 영구 보관 + 클라우드 동기화 + 검색
  - 즐겨찾기 무제한
  - 광고 제거 (만약 광고 도입 시)

### Phase 4 (18개월~): B2B 사찰 라이센스 ★ 메인 매출
- 사찰별 월 30,000~100,000원 (규모별)
- 신도용 가입 코드 발급 → 무료 액세스
- 사찰 자체 콘텐츠 업로드 (스님 법문 음성, 사찰 행사 안내)
- 1,000개 사찰 × 평균 5만원 = **월 5,000만원** 잠재 매출

## 무료 vs 유료 분리

### 무료 (영원히)
- **108배 음성 가이드 무제한** ← 핵심
- 명상·염불 무제한
- 마음기록 30일 보관 (로컬)
- 일일 한마디 1장 + 시간대 추천
- AI 법현 스님 **1일 3회**
- 반야심경 낭독 1편
- 수행자 종 (마음챙김 알림)
- 즐겨찾기 (로컬)

### 멤버십 (월 4,900원 / 연 39,000원)
- AI 법현 스님 **무제한** ← 가장 강력한 결제 유도점
- 모든 경전 낭독 (4명 인도자별)
- 명상 가이드 라이브러리
- 마음기록 영구 + 클라우드 동기화
- 즐겨찾기 영구 + 클라우드 + 검색
- 광고 제거

### 단품 (1회 결제, 영구)
- 사운드 팩 1,900원
- 특별 경전 1편 2,900원

### 보시
- 자유 금액 (1,000원~)
- 1회 또는 정기

## AI 비용 — 결제 유도점이자 운영 비용

### 비용 구조
- Gemini 2.5 Flash: 입력 $0.075/1M tokens, 출력 $0.3/1M tokens
- 평균 1대화 = 입력 200자 + 출력 500자 = 약 1,500 tokens = **$0.0006 (약 0.8원)**
- 월 1,000명 × 평균 5회 = 5,000회 × 0.8원 = **월 4,000원**
- 월 10,000명 × 평균 5회 = 50,000회 = **월 4만원**

### AI 무료 한도 = 결제 유도점
- 1일 3회 무료 → 더 쓰고 싶은 사용자 = 결제 의향 가장 높음
- "오늘 3번 다 썼어요. 멤버십 가입 시 무제한"
- AI 호출 비용 자체는 큰 부담 X, 다만 결제 유도력 확보

## 월별 손익 시뮬레이션

### 1년 후 (사용자 5,000명, conversion 2%)

|---|---|

| 단품 결제 50건 × 평균 2,500원 | +12만원 |
| 보시 평균 200건 × 평균 3,000원 | +60만원 |
| **수익** | **+121만원/월** |
| Vercel Pro | -3만원 |
| Supabase Pro | -3만원 |
| Gemini API | -2만원 |
| Google TTS / OpenAI | -1만원 |
| 토스 수수료 (3%) | -3만원 |
| **운영비** | **-12만원/월** |
| **순수익** | **+109만원/월** |

| 항목 | 금액 |
|---|---|

## 차별화 포인트 (다른 명상 앱이 못 함)

1. **AI 법현 스님 챗봇** — 코끼리·마보엔 없음
2. **108배 음성 가이드 4명 인도자** — 이미 구현
3. **경전 낭독 인도자별** — Phase 2 (Google TTS 그대로 활용)
4. **사찰 라이센스 (B2B)** — 진짜 차별화, 한국 시장 특화
5. **Voice Cloning** — Phase 5 (ElevenLabs로 진짜 스님 톤 클론)
